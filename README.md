# Multilayer Perceptrons (MLP)
A multilayer perceptron (MLP) fully connected neural network using PyTorch with the MNIST, Fashion-MNIST, and
Kuzushiji datasets, achieving over 98%, 92%, and 90% accuracy respectively.
